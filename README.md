## Turbochat

### Description

Simple chat with Hotwire, Turbo and Stimulus.

Implemented in Ruby 3.0 using Rails 7.0 and PostgreSQL.

### Launching

1. Download or clone repo. Install all dependencies and prepare database

```bash
$ bin/setup
```

2. Start server

```bash
$ bin/dev
```
